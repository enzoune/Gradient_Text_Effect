# 🎨 Interactive Gradient Text Effect
> Um componente de tipografia dinâmica desenvolvido com HTML5 e CSS3, apresentando um efeito de preenchimento em gradiente e interatividade em tempo real.

## 📖 Sobre o Projeto

Este projeto faz parte do meu portfólio de estudos em Programação Full-Stack. O objetivo principal foi explorar a estilização avançada de textos, utilizando técnicas de máscara de fundo (background-clip) para criar fontes com cores vibrantes que fogem do padrão sólido.

O grande diferencial deste componente é a sua natureza editável. Diferente de um texto estático, o usuário pode interagir diretamente com o elemento na tela, permitindo visualizar como o gradiente se comporta com diferentes palavras ou extensões de frase.

## ✨ Destaques e Diferenciais Técnicos

* **Manipulação de Background:** Aplicação de gradientes lineares ou radiais mascarados ao texto, demonstrando domínio sobre propriedades modernas de renderização do CSS.

* **Arquitetura Minimalista:** O projeto foca na eficiência, utilizando apenas uma estrutura básica de `div` com classe específica (`.text`), facilitando a integração em qualquer cabeçalho ou banner de sites futuros.

* **Organização Profissional:** O código segue os padrões de engenharia de software com a separação total entre a estrutura (HTML) e a identidade visual (CSS), mantendo a folha de estilos em um diretório dedicado (`css/style.css`).

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Implementação de atributos de edição em tempo real.

* **CSS3:** Estilização tipográfica e máscaras de gradiente.
